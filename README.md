# Simple Calculator in C

## Overview
This project is a simple calculator written in C that performs basic arithmetic operations using operator-based user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero validation
- Invalid operator handling

## Technologies Used
- C programming language
- Dev-C++

## How It Works
The program accepts two numbers and an arithmetic operator from the user, then performs the selected operation using a `switch` statement and displays the result.

## Example
### Input
10
+
5

### Output
Result: 15.00

## File Included
- `calculator.c`

## Learning Outcome
This project helped in understanding:
- basic C syntax
- user input handling
- conditional logic using `switch-case`
- error handling in C
